Join Filter:

1. If the value is the list **['a', 'b', 'c']**, the output will be \ the string **"a - b - c"**.

{% verbatim %} {{ list1 | join:" - " }} {% endverbatim %} : {{ list1 | join:" - " }}.


2. If the value is the list **['python', 'is', 'fun']**, the output will be \ the string **"python - is - fun"**.

{% verbatim %} {{ list2 | join:" - " }} {% endverbatim %} : {{ list2 | join:" - " }}.


3. If the value is the list **['Monday', 'Tuesday', 'Wednesday']**, the output will be \ the string **"Monday // Tuesday // Wednesday"**.

{% verbatim %} {{ list3 | join:" // " }} {% endverbatim %} : {{ list3 | join:" // " }}.

Date Filter:

1. If value is a datetime object (for example, the result of datetime.datetime.now()) OR 2022-12-12T10:30:00.000123, the output will be the string '01/12/2023'.

{% verbatim %} {{ dateTimeStr | date:"SHORT_DATE_FORMAT" }} {% endverbatim %} : {{ dateTimeStr | date:"SHORT_DATE_FORMAT" }}.


2. If value is a datetime object (for example, the result of datetime.datetime.now()) OR 2022-12-12T10:30:00.000123, the output will be the string 'Dec. 12, 2022'.

{% verbatim %} {{ dateTimeStr | date }} {% endverbatim %} : {{ dateTimeStr | date }}.


3. If value is a datetime object (for example, the result of datetime.datetime.now()) OR 2022-12-12T10:30:00.000123, the output will be the string "Sun 12 Jan 2023".

{% verbatim %} {{ dateTimeStr | date:"D d M Y" }} {% endverbatim %} : {{ dateTimeStr | date:"D d M Y" }}.

Default Filter:

1. If the value is " " (the empty string), the output will be "an empty string".

{% verbatim %} {{ emptyStr | default:"an empty string" }} {% endverbatim %} : {{ emptyStr | default:"an empty string" }}.


2. If the value is " " (the empty string), the output will be "This is a default value".

{% verbatim %} {{ emptyStr | default:"This is a default value" }} {% endverbatim %} : {{ emptyStr | default:"This is a default value" }}.

Add Filter:

1. If the value is 10, then the output will be 15. This filter is used to increment a variable in Django templates by adding a constant value to the variable.

{% verbatim %} {{ num | add:"5" }} {% endverbatim %} : {{ num | add:"5" }}.


2. If the value is [1, 2, 3], then the output will be [1, 2, 3, 4, 5, 6]. This filter is used to increment a variable in Django templates by adding a constant value to the variable.

{% verbatim %} {{ list4 | add:"[4, 5, 6]" }} {% endverbatim %} : {{ list4 | add:"[4, 5, 6]" }}.

Capfirst Filter:

1. If the value is “earthly”, the output will be “Earthly”.

{% verbatim %} {{ st1 | capfirst }} {% endverbatim %} : {{ st1 | capfirst }}.


2. If the value is “django”, the output will be “Django”.

{% verbatim %} {{ st2 | capfirst }} {% endverbatim %} : {{ st2 | capfirst }}.

Cut Filter:

1. If the value is “Python is Fun”, the output will be “PythonisFun”, the filter has cut out spaces from the value.

{% verbatim %} {{ st3 | cut:" " }} {% endverbatim %} : {{ st3 | cut:" " }}.


2. If the value is “January - February - March”, the output will be “January February March”, the filter has cut out spaces from the value.

{% verbatim %} {{ st4 | cut:"-" }} {% endverbatim %} : {{ st4 | cut:"-" }}.

Dictsort Filter:

1. If the value is: 

                    [
                        {'name': 'Josh', 'age': 19},
                        {'name': 'Dave', 'age': 22},
                        {'name': 'Joe', 'age': 31},
                    ]
Then the output would be: 
                    [
                        {'name': 'Dave', 'age': 22},
                        {'name': 'Joe', 'age': 31},
                        {'name': 'Josh', 'age': 19},

                    ]


{% verbatim %} {{ list5 | dictsort:"name" }} {% endverbatim %} : {{ list5 | dictsort:"name" }}.

Timesince Filter:

1. If publication_date is a date instance representing midnight on 1 July 2024, and comment_date is a date instance for 09:00 on 1 July 2024, then the output would be “9 hours 9 minute”.

{% verbatim %} {{ publication_date | timesince:comment_date }} {% endverbatim %} : {{ publication_date | timesince:comment_date }}.

2. If publication_date is a date instance representing midnight on 1 July 2024, and without the argument, the comparison point is now.

{% verbatim %} {{ publication_date | timesince }} {% endverbatim %} : {{ publication_date | timesince }}.

Truncatechars Filter:

1. If the value is “Moe is a slug”, the output will be “Moe is…”.

{% verbatim %} {{ st5 | truncatechars:7 }} {% endverbatim %} : {{ st5 | truncatechars:7 }}.

Wordcount Filter:

1. If the value is “Moe is a cat”, the output will be 4.

{% verbatim %} {{ st6 | wordcount }} {% endverbatim %} : {{ st6 | wordcount }}.

Truncatewords Filter:

1. If the value is "You can build web apps with python and Django", the output will be "You can build web...".

{% verbatim %} {{ st7 | truncatewords:4 }} {% endverbatim %} : {{ st7 | truncatewords:4 }}.
