These filtering from GeeksforGeeks

### It is used to add an argument to the value.
If value is 18, then the output will be 20 {% verbatim %} {{ value=20 | add:"2" }} {% endverbatim %} : {{ num | add:"2" }}.

### It is used to add slashes before quotes. Useful for escaping strings in CSV..
If value is “I’m Jai”, the output will be “I\’m Jai”. {% verbatim %} {{ value | addslashes }} {% endverbatim %} : {{ st | addslashes }}.

### It is used to capitalize the first character of the value. If the first character is not a letter, this filter has no effect.
If value is “jai”, the output will be “Jai”. {% verbatim %} {{ value | capfirst }} {% endverbatim %} : {{ st2 | capfirst }}.

### It is used to center the value in a field of a given width.
If value is “Jai”, the output will be ” Jai “. {% verbatim %} {{ value | center:"15" }} {% endverbatim %} : {{ st3 | center:"15" }}.

### It is used to remove all values of arg from the given string.
If value is “String with spaces”, the output will be “Stringwithspaces”. {% verbatim %} {{ value | cut:" " }} {% endverbatim %} : {{ st4 | cut:" " }}.

### It is used to format a date according to the given format.
If value is a datetime object (e.g., the result of datetime.datetime.now()), the output will be the string ‘Thu 06 Feb 2020’. {% verbatim %} {{ value | date:"D d M Y" }} {% endverbatim %} : {{ dateTime | date:"D d M Y" }}.

### It is used to give a default value to a variable. If variable evaluates to False, it will use the default argument given else the variable value itself.
If value is “” (the empty string), the output will be nothing. {% verbatim %} {{ value | default:"nothing" }} {% endverbatim %} : {{ st6 | default:"nothing" }}.

### It takes a list of dictionaries and returns that list sorted by the key given in the argument.
If value is: 
                [
                    {'name': 'zed', 'age': 19},
                    {'name': 'amy', 'age': 22},
                    {'name': 'joe', 'age': 31},
                ]
then the output would be: 
                [
                    {'name': 'amy', 'age': 22},
                    {'name': 'joe', 'age': 31},
                    {'name': 'zed', 'age': 19},
                ]
{% verbatim %} {{ value | dictsort:"name" }} {% endverbatim %} : {{ dict | dictsort:"name" }}.

### It returns True if the value is divisible by the argument.
If value is 21, the output would be True. {% verbatim %} {{ value | divisibleby:"3" }} {% endverbatim %} : {{ num2 | divisibleby:"3" }}.

### It is used to format the value like a ‘human-readable’ file size (i.e. ’13 KB’, ‘4.1 MB’, ‘102 bytes’, etc.).
If value is 123456789, the output would be 117.7 MB. {% verbatim %} {{ value | filesizeformat }} {% endverbatim %} : {{ fileSize | filesizeformat }}.

### It is used to return the first item in a list.
If value is the list [‘a’, ‘b’, ‘c’], the output will be ‘a’. {% verbatim %} {{ value | first }} {% endverbatim %} : {{ list1 | first }}.

### It is used to join a list with a string, like Python’s str.join(list)
If value is the list [‘a’, ‘b’, ‘c’], the output will be the string “a // b // c”. {% verbatim %} {{ value | join:" // " }} {% endverbatim %} : {{ list1 | join:" // " }}.

### It is used to return the last item in a list.
If value is the list [‘a’, ‘b’, ‘c’], the output will be the string 'c' {% verbatim %} {{ value | last }} {% endverbatim %} : {{ list1 | last }}.

### It is used to return the length of the value. This works for both strings and lists.
If value is [‘a’, ‘b’, ‘c’] or “abc”, the output will be 3. {% verbatim %} {{ value | length }} {% endverbatim %} : {{ list1 | length }}.

### It is used to display text with line numbers.
If value is: 
                one
                two
                three
the output will be: 
                1. one
                2. two
                3. three
{% verbatim %} {{ value | linenumbers }} {% endverbatim %} : {{ st7 | linenumbers }}.

### It is used to converts a string into all lowercase.
If value is My Name is Jai, the output will be my name is jai. {% verbatim %} {{ value | lower }} {% endverbatim %} : {{ st8 | lower }}.

### It is used to return the value turned into a list. For a string, it’s a list of characters. For an integer, the argument is cast to a string before creating a list.
If value is the string “Naveen”, the output would be the list [‘N’, ‘a’, ‘v’, ‘e’, ‘e’, ‘n’]. If value is 123, the output will be the list [‘1’, ‘2’, ‘3’]. {% verbatim %} {{ value | make_list }} {% endverbatim %} : {{ st9 | make_list }}.

### It is used to return the value turned into a list. For a string, it’s a list of characters. For an integer, the argument is cast to a string before creating a list.
If value is the list [‘a’, ‘b’, ‘c’, ‘d’], the output could be “b”. {% verbatim %} {{ value | random }} {% endverbatim %} : {{ list1 | random }}.

### It is used to return a slice of the list.
If some_list is [‘a’, ‘b’, ‘c’], the output will be [‘a’, ‘b’]. {% verbatim %} {{ value | slice:":2" }} {% endverbatim %} : {{ list1 | slice:":2" }}.

### It is used to convert to ASCII. It converts spaces to hyphens and removes characters that aren’t alphanumerics, underscores, or hyphens. Converts to lowercase. Also strips leading and trailing whitespace.
If value is “ Jai_/ is a slug ”, the output will be “jai-is-a-slug”. {% verbatim %} {{ value | slugify }} {% endverbatim %} : {{ list1 | slugify }}.

### It is used to format a time according to the given format.
If value is equivalent to datetime.datetime.now(), the output will be the string “01:23”. {% verbatim %} {{ value | time:"H:i" }} {% endverbatim %} : {{ dateTime | time:"H:i" }}.

### It is used to convert a string into titlecase by making words start with an uppercase character and the remaining characters lowercase. This filter makes no effort to keep “trivial words” in lowercase.
If value is “my FIRST post”, the output will be “My First Post”. {% verbatim %} {{ value | title }} {% endverbatim %} : {{ st5 | title }}.

### It is used to recursively take a self-nested list and returns an HTML unordered list – WITHOUT opening and closing
    tags.
    if var contains [‘States’, [‘Kansas’, [‘Lawrence’, ‘Topeka’], ‘Illinois’]], then {{ var|unordered_list }} would return:
    {% verbatim %} {{ value | unordered_list }}: {% endverbatim %}

### It is used to convert a string into all uppercase.
If value is “My Name is Jai”, the output will be “MY NAME IS JAI”. {% verbatim %} {{ value | upper }} {% endverbatim %} : {{ st8 | upper }}.

### It is used to return the number of words.
If value is “jai is a slug”, the output will be 4. {% verbatim %} {{ value | wordcount }} {% endverbatim %} : {{ st8 | wordcount }}.
